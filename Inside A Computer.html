Insight

How Data
Moves Between
Components

  • The CPU sends instructions at up to 128 GB/s over the memory bus, fetching data from RAM every few nanoseconds to sustain peak performance.
  • The CPU offloads rendering workloads to the GPU via PCIe 5.0, which provides over 128 GB/s of bidirectional bandwidth.
  • RAM acts as a staging area — data is loaded from storage into RAM first, so the CPU can access it thousands of times faster than from disk.
  • NVMe SSDs communicate via the PCIe lane directly, achieving read speeds of up to 14,000 MB/s — 30× faster than traditional SATA SSDs.
  • The Network Interface Card (NIC) translates data packets and manages the TCP/IP stack, routing internet traffic through the system bus.

CPU
Central Processing Unit — The System Brain

The CPU executes the fetch-decode-execute cycle billions of times per second. Modern CPUs use superscalar pipelines, branch prediction, out-of-order execution, and multi-level caches (L1, L2, L3) to maximize throughput. With up to 64 cores in consumer chips, they handle complex sequential and parallel workloads simultaneously.

RAM
Random Access Memory — The Workspace

RAM is volatile memory — it loses all data when power is cut. It sits between the slow disk and the tiny-but-fast CPU caches, acting as the workspace for active programs. DDR5 memory uses a dual-channel 64-bit bus and can sustain bandwidth of 96+ GB/s per channel.

Non-Volatile Memory — The Long-Term Vault

Unlike RAM, storage retains data permanently. Modern NVMe SSDs use PCIe Gen 5 lanes to achieve sequential read speeds up to 14,000 MB/s. HDDs use spinning magnetic platters and are slower but cheaper per gigabyte. SSDs use NAND flash cells — stacking 200+ layers in 3D structures.

NETWORK
Network Interface Card — The Global Connector

The NIC manages all communication between the computer and external networks. It handles MAC addressing, packet framing, and protocol translation. WiFi 7 (802.11be) can achieve throughput up to 46 Gbps using Multi-Link Operation and 320 MHz channels. Modern NICs offload TCP/IP stack processing from the CPU.
